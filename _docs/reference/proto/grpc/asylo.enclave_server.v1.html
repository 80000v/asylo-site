ServerCommand

A command provided to the enclave server.

Name Description
UNKNOWN
INITIALIZE_SERVER

Indicates that the enclave should build and prepare the gRPC server (e.g. determine the final server address). This does not actually launch the server.
RUN_SERVER

Indicates that the enclave should start the gRPC server.

ServerConfig

Represents an enclave gRPC server’s configuration.

Field Type Description
host string

The host to run on in the form of IPv6 address minus the port.
port int32

The port to run on. A port of 0 indicates that the port should be auto-selected by the system.
